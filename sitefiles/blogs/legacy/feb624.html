bloodbound

"what is bloodbound????" bloodbound is a msp(fan)a that i'm working on with a friend of mine! it's currently in a development stage, but i've been the main artist so far :p
bloodbound is also a kind of testing ground for php. instead of having to update the html of each page individually, i just have to make a new entry in my php database, upload a few asset files, and bam! new page!
originally, i worked on the web design in only html/css/js... no php. this proved to be horrible when i wanted to implement something like a go back button! i'd have to update the (at the time) 40~ webpages individually once i got the css finalized :( this drove me away from the project for a while... until one day i randomly decided "man i should learn php" because i heard that mspa used it, and that it could really help with developing something like this. i then whipped up a proof of concept mspa-like thing locally with xampp, just to see if it was even possible for me to do it... which it was! once i finished it, i explored hosting options
initially, i tried hosting with a2. this... didn't really suit my needs! i wasn't able to find the htdocs folder, and good god it was expensive. sure, it was like 12 dollars a month... but you had to pay that upfront. paying over 100 dollars a year for something i wouldn't use the full potential for didn't really sound like a good deal?
after this i tried looking at more options... but after a2 burnt me, i was scared off by hostinger being paid only. i then looked up "php hosting free" and found infinitlyfree! yes it's spelled wrong



i then set up the mspa test on the server easily enough, and started work on php-ifying bloodbound :p
