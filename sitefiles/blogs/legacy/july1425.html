hey! so! it's been a month!

i have succumbed to the i3 mind virus. AND I'M COMING FOR YOU NEXT!!!!!!!!!!!

i've spent the last month running linux as my main pc os, and it's been fun! i've been learning a lot about how linux works under the hood, and many apps (and even games!) are preforming better than i expected :D
you may notice that i'm not on manjaro anymore! that is because manjaro made like any other linux distro i've used, and decided to crap out and show me nothing but a black screen. that was really fun
so i did what any sane person would do, and move to a distro that lets you have more control over what is and isn't installed on your computer! arch!

arch's design is based around only installing the packages you need. need a drawing tablet driver? desktop environment? base firefox? gui wifi manager? nano? no??? then you don't have to install it!! ^_^
this is really good for my usecase, as i don't have to worry as much about system stability. if i install something fucky, i'll most likely know what's messing up my system! then, i can just remove it with the live install dvd :D it truly is oh so very shrimple
i prefer this a lot to the "oh just reinstall your os" tactic i've used since i got my first pc that i could really mess with, as i get to keep my data that's on my boot hard drive!
while arch's install is something you have to do via the cli (and the wizard that it comes with is really fucky), the arch wiki has a very easy to follow install guide that will help people who don't know anything about the commands they're pumping into their pc :o i'd say part of the reason my venture to linux has been so successful is because i'm learning about how to configure my system, instead of just installing shit and hoping it works

speaking of configuration...
when i first got arch up and running functionally on my pc, the first thing i did was install kde!

it ran pretty damn well, too. i was able to make it pretty quickly with the knowledge i had gained with my manjaro rice, and i had fun messing around with some of the software included with the full kde suite for a bit!
However!

wine was the death of me once again!

like. like are you. are. are you seeing this shit

this pissed me off so bad i ended up switching to i3
looking back, i'd guess now that this is more of an xwayland thing than a kde-specific thing. i may be a little stupid
however, it's too late for me to change back to kde! navigating windows with vim bindings is actually insane and i love it
i use i3-gradients, which is a fork of i3! i chose it because i thought it would support vertical gradients... but it doesn't yet :(

i really like i3 because it allows for keyboard window navigation, in-depth configuration, and because it's generally quick and responsive! i use polybar for my status bar, as the default i3 one is. ugly

anyways, that's pretty much it! see you next year
