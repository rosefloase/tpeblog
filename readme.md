## setting up
### linux
1 - prerequisites:\
[nodejs](https://nodejs.org/), [npm](https://www.npmjs.com/), and [zsh](https://www.zsh.org/)

2 - installing and running 

install ``@11ty/eleventy-plugin-syntaxhighlight`` and ``luxon`` with npm  
run ``runserver.zsh`` with ``zsh``
